<!DOCTYPE html>                 -> Defines that this document is an HTML5 document
<html>                          -> Root element of an HTML page
<head>                          -> Contains meta information about the HTML page
<title>Page Title</title>       -> Specify a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
</head>                         -> End tag for Head
<body>                          -> Define the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

<h1>My First Heading</h1>       -> Define a large heading
<p>My first paragraph.</p>      -> Define a paragraph

</body>
</html>
